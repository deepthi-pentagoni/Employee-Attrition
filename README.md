# Employee-Attrition
## Overview
This project predicts whether an employee is likely to leave the company using Machine Learning. The analysis includes data preprocessing, exploratory data analysis (EDA), model training, evaluation, and business recommendations.

## Dataset
- IBM HR Analytics Employee Attrition Dataset
- Source: Kaggle
- Records: 1,470 employees

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Machine Learning Models
- Logistic Regression
- Random Forest Classifier
- Gradient Boosting Classifier

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- ROC-AUC Score
- Confusion Matrix
